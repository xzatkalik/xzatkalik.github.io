Tiny PXE Server Settings

Running the Tiny PXE Server executable (pxesrv.exe) will start the program with default settings which will then need to be edited as required - e.g. a Boot File will not be specified and will need to be added manually.

Having to manually input the settings every time Tiny PXE Server is started can be frustrating and tedious, so it's also possible to use a configuration file. Right-click within the main window and select the "Save settings to config.ini" option to create a configuration file using the current settings.

It's also possible to specify settings in the configuration file config.ini.

  • GUI Mode
  • Config.ini Settings

NOTE - to apply any changes made in the UI whilst Tiny PXE Server is running you will need to stop (Offline) and (re)start (Online) services.
